# State

This section about state.

```tsx
import { init } from "@blaze";

const Hello = function () {
    const { render, state } = init(this);
    state(null, {
        name: "ferdiansyah",
    });

    const change = () => (this.state.name = "safina sahda");

    render(() => <p>{this.state.name}</p>);
};
```
